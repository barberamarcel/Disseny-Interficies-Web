Les meves habilitats

Disseny Web

Com a programador front-end, puc crear i implementear una interfície d'usuari d'una pàgina web o una aplicació web. Això involucra treballar amb HTML i CSS per a dissenyar i crear pàgines web.

Python

a desenvolupador de Python, puc dissenyar i desenvolupar aplicacions de software utilitzant el llenguatge de programació Python. Això pot involucrar crear aplicacions que requereixin d'algun script i implementació d'algoritmes, depenent de cada aplicació.

Ethical Hacking

De la poca experiència la qual tinc en el món del Hacking Ètic, he tingut el plaer d'aprendre "SQL Injection" les quals algunes s'han fet mitjançant scripts programats en Python. També he pogut aprendre les tècniques de SSRF, CSRF, XSS, XXE, XML Injection i Directory Traversal. Moltes d'elles dutes a terme en sistemes operatius Linux.
